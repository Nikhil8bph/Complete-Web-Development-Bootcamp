Flex box in css

Flex Direction Types:

  • flex-direction:row-This is default value
  • flex-direction:Column-this will set the elements column wise.
  • flex-direction:row-reverse- This will start filling the elements from right side instead of left
  • flex-direction:column-reverse- This will put the elements in reverse column.

Flex Wrap Property:

  • Flex-wrap:nowrap- This is default value.
  • Flex-wrap:wrap- This will wrap the flex container items according to the sixe of the screen.
  • Flex-wrap:wrap-reverse- This will wrap the text in reverse direction.

Flex Flow Property

This property is used to set bot flex direct and wrap. For example: flex-flow: row-reverse wrap;
