Mongo DB Basics

  • Tables are called collections
  • Rows are called documents

    Commands

  • show dbs: Shows the list of database created.
  • use < db_name > : Craetes a new db with the mentioned name or switch to the db if its already there.
  • db.< collection_name >: creates a collection/table with that name.
  • db.< collection_name >.insert({}) : Inserts a document/row into the collecion
  • show collections: Shows all the tables/collections of the db that is in use

Inserting data in collections

  • Create DB: use Employee
  • Insert one records: db.details.insertOne({name:"Puja",ID:"107",Job:"Engineer"})
  • Show the documents/rows of db : db.details.find()
  • Insert many records: db.details.insertMany([{name:"Puja",ID:"107",Job:"Engineer"},{name:"Sudhanshu",ID:"108",Job:"Engineer"},{name:"Jit",ID:"109",Job:"Engineer",Salary:200000}])
