CSS Selectors

CSS has following types of selectors

  • Element Selector
  • ID Selector
  • Class Selector
  • Group Selector

Element Selector

It targets all elements of a particular type

<style> p{ color: chocolate; } </style>

ID Selector

It targets the element of particular ID

<style> p{ color: chocolate; } #firstPara{ color: cornflowerblue; } </style>

Class Selector

It targets the element of particular Class.One element can take the property of multiple classesss

<style> .icolor{ color: rebeccapurple; } .bgColor{ background-color: darkkhaki; } </style>

Group Selector

It targets group of elements.

<style> div,h4{ color: salmon; } </style>
