CSS Fonts

Following are some of the properties of CSS fonts

  • Font-family: It can be of 2 types:
    1. Web Safe: Can be displayed by any browser even if not available with the OS
    2. Web: Imported from internet with the help of links that is they are available on web server
  • Font-size: Helps in adjusting the size of the font
  • Font-weight: For example bold
  • Font-style
  • Line-height: Provides the spacing between each line
