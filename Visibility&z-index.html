Visibility and z-index

  • Visibility: By default it is set as visible. If we set it as hidden the element will be hidden leaving behind an empty space at its place
  • Display:none-If we set display as none for an element, then the element will be hidden. Unlike visibility:hidden property it does not leave behind an empty space
  • Z-index: It will work only if elements are given positions other than static. If the elements are stacked then the element with maximum z-index comes at the top
