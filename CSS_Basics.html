Cascading Style Sheet

CSS stands for cascading style sheet. It is used to design the web pages and make them responsive that is our web page will adjust itself according to mobile or desktop or other devices.

CSS can be included with HTML in three ways

  • Inline: CSS is added to the elements directly using style attribute
  • Internal: CSS is kept under head tag using style tag
  • External: CSS is included in seperate .css file and included in HTML

Inline CSS:

<p style="colour :brown;">
