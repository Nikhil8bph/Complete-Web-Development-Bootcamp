Position Property

CSS has 4 types of positions:

  • Static: By default all elements are in static position.
  • Absolute: The element will be shifted with respect to the position of parent element
  • Relative: The element will shift from its normal position according to the configuration given leaving behind a gap at its original position.
  • Sticky: The element will stick at its position even if the page is scrolled
  • Fixed: It will fix an element at its mantioned position even if the page is scrolled. If no dimension is given for the position the element will by default come at tol left corner. But in sticky it will remain at its original positon and it dont require position dimensions
